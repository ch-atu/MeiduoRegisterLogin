## 登录注册

- tips: 本套登录注册系统涵盖发送短信验证码、多用户登录、jwt认证登录
- 基于djangorestframework、vue2
- 只是简单的实现vue功能，最终配合脚手架进行开发
### 1.依赖
>pip install -r requirements.txt

### 2.执行迁移文件
> python manage.py makemigrations
> 
> python manage.py migrate
